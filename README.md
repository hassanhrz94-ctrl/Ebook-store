# 📚 Book Store Web App

A modern and responsive **Book Store web application** built with **Next.js, Tailwind CSS, and Better Auth**. Users can browse books, explore categories, and securely authenticate to access personalized features.

## 🚀 Live Demo

👉 https://ebook-store-eta.vercel.app/

## ✨ Features

* 🔐 Authentication system (Better Auth)
* 📚 Browse and explore books
* 🔎 Category-based filtering
* ⚡ Fast and responsive UI
* 🎨 Clean design using Tailwind CSS
* 🌐 Fully deployed on Vercel

## 🛠️ Tech Stack

* **Frontend:** Next.js
* **Styling:** Tailwind CSS
* **Authentication:** Better Auth
* **Deployment:** Vercel

## 📦 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/book-store.git
cd book-store
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
